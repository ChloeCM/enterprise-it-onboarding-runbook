# Scenario SC-002 - New Starter Msiing Application Access

## Situation

New employee can log into their device and email but cannot access a required missing application.

## First-line Checks

- Confirm user identity, role and location
- Confrim device login and Microsoft 365 access are working
- Verify which application access is missing
- Confirm whether accesss request has been submitted or approved

## Outcome

Issue identified as missing application access. Escalation is required to the appropriate application team with user details and access requirements.
