# New-Joiner IT Onboarding & Access Runbook

## Scope

This runbook applies to first-line IT support assisting new employees during onboarding.

## Assumptions

- User has a confirmed start date
- HR has completed contract setup
- First-line IT does not assign application roles

## Day 0 vs Day 1 Responsibilities

- HR responsibilities
- IT responsibilities
- Line manager responsibilities

## First-Line IT Onboarding Checklist

- Device access
- Account login
- Microsoft 365 access
- Basic application access verification

## User Types

- Office users
- Plant users
- Lab users

## Common Onboarding Issues

- Missing access
- Incorrect role or location
- Device issued before account setup

## What First-Line IT Can Resolve

- Account unlocks
- Password resets
- Basic access verification

## What Must Be Escalated

- Application role assignment
- SAP access
- Vendor or lab systems

## Escalation Information Required

- User details
- Location and role
- Screenshots or error messages
